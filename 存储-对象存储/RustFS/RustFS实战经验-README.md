## 关于 RustFS

[RustFS](https://github.com/rustfs/rustfs) 是一个用 Rust 编写的高性能对象存储服务，兼容 S3 协议。在私有化部署时，需要特别注意其对 S3 标准的**严格遵循**，这与国内云厂商（OBS/COS/OSS）的**兼容性放宽**策略有所不同。

### 关键差异点

| 特性 | RustFS | OBS/COS/OSS |
|------|--------|-------------|
| S3 标准遵循 | 严格 | 宽松（兼容） |
| 预签名 URL Host 校验 | 严格 | 可能宽松 |
| 分片最小大小 | 5MB（标准） | 100KB-1MB（放宽） |

---

## 最佳实践

1. **始终用域名生成预签名 URL**，不要在生成后做字符串替换
2. **分片大小 ≥ 5MB**（最后一片除外），不依赖厂商兼容性
3. **多云环境测试**，不要只在一家云厂商测试通过就上线
